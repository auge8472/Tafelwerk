Einleitung

Einige klärende Worte

Diese Sammlung von Formeln, geometrischen Konstruktionen und Ähnlichem ist aus privatem Interesse entstanden und wird auch so gepflegt. Sie erhebt folglich keinen Anspruch auf ständige redaktionelle Begleitung oder darauf, sämtliche Wissensgebiete abzudecken, geschweige denn darauf, vollständig zu sein.

technische Hinweise

Einige Formelzeichen und Einheiten werden durch Buchstaben des griechischen Alphabets dargestellt. Dafür werden hier HTML-eigene Namen benutzt. Diese werden von Browsern, die nur HTML bis Version 3.2 darstellen können, nicht erkannt. Die bekannten graphischen Browser ab Version 4 können HTML 4 darstellen. Weiterhin muß auf dem System des Nutzers eine entsprechende Schriftart installiert sein, damit der Browser die Zeichen darstellen kann.
